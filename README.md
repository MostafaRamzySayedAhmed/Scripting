# Scripting
It's a Full Exercise on Bash/Batch Scripting.

**Guessing Game**
* It's a Bash Script That Will Continuously Ask Users to Guess The Number of Files in Current Directory Till Picking The Correct One.
* The User Will Be Informed if His Picked Number is Too High or Too Low.
* Once The Correct Number is Guessed, The User Will be Congratulated.

**System Info**
* It's a Batch Script That Will Query Different System Information, Then Group Them into Three Different Categories: "Windows", "Hardware" & "Network" Info.
